# linear-regression-from-scratch
Implementation of Simple Linear Regression in Python using NumPy and Matplotlib for predictive analysis and visualization.

## Features
- Data preprocessing
- Training a Simple Linear Regression model
- Predicting output values
- Visualization using Matplotlib
- Beginner-friendly implementation

## Technologies Used
- Python
- NumPy
- Matplotlib
- Jupyter Notebook
